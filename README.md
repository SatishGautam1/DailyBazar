# DailyBazar 🛒

DailyBazar is a modern Android application tailored for everyday shopping and home needs. Designed with a clean, light-mode-exclusive UI, it provides a seamless user experience from the moment the app opens.

## ✨ Features
* **Custom Splash Screen:** Brand-focused splash experience utilizing the modern Android 12+ Splash Screen API and custom themes.
* **Modern Authentication UI:** Clean, intuitive Login and Sign-Up screens featuring Material Design 3 components.
* **Edge-to-Edge Design:** Beautiful status bar integration with overlapping brand headers and floating cards.
* **Light-Mode Exclusive:** Locked to a crisp, light theme to maintain strict brand consistency.

## 🛠 Tech Stack
* **Language:** Java
* **UI toolkit:** XML, Material Design 3 (Material Components for Android)
* **Architecture:** Traditional Activity-based layout with modern View handling.
* **Minimum SDK:** 24 (Android 7.0)

## 🚀 Getting Started
To build and run this project locally:
1. Clone the repository: `git clone https://github.com/SatishGautam1/DailyBazar.git`
2. Open the project in **Android Studio**.
3. Sync project with Gradle files.
4. Build and run on an emulator or physical device.
